## 1. 赖鑫——架构 & 认证 & 文档

### 后端文件

| 模块             | 文件路径                                | 责任                                              |
| ---------------- | --------------------------------------- | ------------------------------------------------- |
| 应用入口         | `backend/app/main.py`                   | 创建 FastAPI 实例、挂 `api_router`                |
| 路由汇总         | `backend/app/api/v1/api.py`             | 统一 include 各 endpoints                         |
| 配置             | `backend/app/core/config.py`            | 读取环境变量、数据库配置、JWT 配置等              |
| 安全             | `backend/app/core/security.py`          | 密码哈希、JWT 生成/校验基础实现                   |
| 依赖             | `backend/app/core/deps.py`              | `get_db`、`get_current_user`、`get_current_admin` |
| DB 会话          | `backend/app/db/session.py`             | 创建 `SessionLocal`、引入 MySQL DSN               |
| DB Base          | `backend/app/db/base.py`                | `Base = declarative_base()` + 导入所有 models     |
| 用户模型         | `backend/app/models/user.py`            | ORM 定义（users 表）                              |
| 用户 Schema      | `backend/app/schemas/user.py`           | `UserBase`、`UserCreate`、`UserRead`              |
| 认证 Schema      | `backend/app/schemas/auth.py`           | `LoginRequest`、`Token`、`TokenData`              |
| 认证接口         | `backend/app/api/v1/endpoints/auth.py`  | `POST /auth/login`、`GET /auth/me`                |
| 管理接口（骨架） | `backend/app/api/v1/endpoints/admin.py` | 先写空壳，后续给 C 做 config 用                   |

### 文档文件

| 文档     | 文件路径               | 责任                                      |
| -------- | ---------------------- | ----------------------------------------- |
| 接口文档 | `docs/api-docs.md`     | 定义各接口 URL、参数、返回结构，主笔      |
| DB 文档  | `docs/db-schema.md`    | 表结构、字段解释、索引说明，和 B/C 一起补 |
| 部署说明 | `docs/deploy-notes.md` | 如何启动后端/前端、环境依赖               |
| 项目说明 | `README.md`            | 项目简介、目录结构、分工说明              |

## 2. 王成友——日志模块（核心一条线）

### 后端文件

| 模块         | 文件路径                               | 责任                                                         |
| ------------ | -------------------------------------- | ------------------------------------------------------------ |
| 日志模型     | `backend/app/models/log.py`            | ORM 定义（logs 表）                                          |
| 日志 Schema  | `backend/app/schemas/log.py`           | `LogCreate`、`LogRead`、`LogQueryParams` 等                  |
| 日志接口     | `backend/app/api/v1/endpoints/logs.py` | `POST /logs`，`GET /logs`，`GET /logs/{id}`，`GET /logs/export`，`POST /logs/upload-file` |
| 日志入库服务 | `backend/app/services/log_ingest.py`   | 文件解析 → 调用 parser → 写入 DB → 调告警引擎                |
| 日志解析工具 | `backend/app/utils/parser.py`          | 不同 source 的行级解析函数                                   |
| CSV 导出工具 | `backend/app/utils/csv_export.py`      | 查询结果生成 CSV、返回流                                     |

| 类型         | 路径                      | 责任                          |
| ------------ | ------------------------- | ----------------------------- |
| 日志格式说明 | `docs/log-format.md`      | 说明你们自定义/选用的日志格式 |
| 样例日志     | `samples/web_logs.txt` 等 | 自造几份样例日志文件用于测试  |
